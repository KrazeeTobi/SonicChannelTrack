カナダ大使館

--------

カナダ・パビリオンでは、カナダから次の14の会社が出展いたします。ATI Technologies Inc.、Enzyme Testing Labs、GestureTek Inc、Autodesk Ltd、Groove Games、Humagade Inc、SilverBirch Studios Inc、SplitFish Gameware Inc、Wave Generation Inc、Bitcasters Inc、Bug-Tracker、Capybara Games、DREAMCATCHER INTERACTIVE INC.、そしてPreview Media。
カナダ・パビリオンで、来日中の担当者と直接お話いただき、各社のゲーミング技術のデモをご覧ください。 また、9月20日（水）にカナダ大使館で開催されるVideo Game Seminarにもご参加ください。カナダ・パビリオンとカナダVideo Game Seminarの詳しい情報に関しては、 http://www.canadanet.or.jp/it/seminar.shtml（カナダ大使館Website）をご参照頂くか、stephane.beaulieu@international.gc.ca（ステファン）まで連絡してください。
