東京ゲームショウ2018 「ソニック」関連情報（09/14 ステージ生放送 情報更新）

2018.08.31

2018年9月20日（木）～23日（日）の期間、千葉県・幕張メッセで開催される東京ゲームショウ2018のセガゲームスブースに、ソニックシリーズ最新作『チームソニックレーシング』のゲームがいち早く体験できるコーナーが出展します。ゲームをご体験いただいた方には、『チームソニックレーシング』特製ステッカーをプレゼントします。

また、会場では本作開発陣が登場するステージイベントを実施、さらに、物販コーナーにはソニックに関連する新商品が登場するなど、様々な催しを予定しています。

出展概要 イベント会場で『チームソニックレーシング』のゲームをご体験いただけます。プレイいただいた方にはノベルティグッズをプレゼントします。また、セガゲームスブースのメインステージでは、本作の開発陣が登壇するステージイベントを実施します。ステージイベントの様子は、インターネット上の生放送番組でもご覧いただけます。
→セガステージ生放送の詳細

ソニック関連ステージイベント

9月21日（金）12:00～12:30

シリーズプロデューサーの飯塚隆と『チームソニックレーシング』ディレクターの河原塚俊介が、『チームソニックレーシング』の実機を用いてゲームの魅力をご紹介します。

登壇者 「ソニック」シリーズプロデューサー / 飯塚 隆
『チームソニックレーシング』ディレクター / 河原塚 俊介

9月22日（土）16:20～16:50

シリーズプロデューサーの飯塚隆と『チームソニックレーシング』リードコンポーザーの瀬上純によるトークショーを開催。ステージの最後には瀬上純によるミニライブを実施します。

登壇者 「ソニック」シリーズプロデューサー / 飯塚 隆
『チームソニックレーシング』リードコンポーザー / 瀬上 純

セガゲームスブース ステージ生放送

セガゲームスブース メインステージの様子を、4日間の期間中、毎日インターネット生放送番組でお届けします。ステージ生放送に関する詳しい情報は、「TGS2018セガ公式特設サイト」でご確認ください。

物販情報

東京ゲームショウ2018物販コーナーでは、『チームソニックレーシング』はもちろん、ソニックに関連する新商品が多数登場します。この機会にぜひお求めください。

  • ※物販ブースはゲーム体験コーナーのあるセガゲームスブースとは異なる場所にありますので、ご来場の際はご注意ください。ゲーム体験コーナーのあるブースでは商品販売は行いません。
  • ※ソニック関連商品の一部は、セガゲームス物販ブースのほか、ライセンス商品お取り扱い各社のブースで販売いたします。販売ブースに関する詳しい情報は東京ゲームショウ2018 公式サイトをご確認ください

ライセンス商品情報

株式会社コスパより各種ソニックグッズが、「東京ゲームショウ2018」にて先行発売決定！詳しい情報は「COSPA | 東京ゲームショウ2018 コスパブース　物販コーナー 出店情報」をご確認ください。

  • ※入場にはチケット（有料）が必要です。
  • ※一般公開日は9/22（土）・9/23（日）の2日間です。一般のご来場の方はビジネスデイにはご入場いただけませんのでご注意ください。
  • ※配布物の数量には限りがあります。なくなり次第終了となりますので予めご了承ください。また、デザインは予告なく変更になる場合があります。
  • ※ゲーム体験コーナー、ステージイベントについて、混雑時には入場制限をもうけさせていただく場合があります。会場ではブーススタッフが誘導いたします。
  • ※ステージイベントの撮影・録音は禁止です。
