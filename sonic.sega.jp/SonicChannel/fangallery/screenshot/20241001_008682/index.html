スクリーンショット

スクリーンショット 2024年10月「小さい！」

2024.10.01

今月もスクリーンショットのコーナーにたくさんの作品をご応募頂き、ありがとうございます！

皆さんから頂いた作品を、スタッフ達からのコメントと合わせてご紹介いたします。

けいこうとう

ちいさくなちゃた！

SONICCDのステージでちいさくなちゃた！

スタッフコメント

小さくなっちゃったソニックが、きょとんとこちらを見ている姿に癒されました！一見コミカルに見えますが、相手を小さくしてしまう狂気の発明...まさにメタリックマッドネスらしい1枚です！

にす

RUN OR RUN

ソニックジェレーションズ白の時空のCity Escape Act1ではソニックがGUNのトラックに追いかけられる場面があります、トラックとソニック大きさが違いすぎない！？

スタッフコメント

G.U.N.の巨大トラックが迫りくる緊迫感が伝わってくる見事な1枚！その巨体を生かして周囲の車を弾き飛ばし、建物を破壊、さらにはルーフから回転刃を出して追ってくる恐怖といったら...。ソニック、逃げてー！

たこやきまん

すっごく小さい

赤いレーザーにあたって小さくなってしまったソニックとテイルス。ACT２のボスでは小さいエッグマンと小さいエミーが出てきますが、、、エミーがちょっと不気味ですね。

スタッフコメント

ソニックとテイルスも、Dr.エッグマン謹製・ミニチュア版エミーに困惑を隠せない様子！抱きついての爆発攻撃を繰り出す、独特な存在感がある敵でしたね。ところでこのエミー、本人が見たらどう思うのか気になります...。

いず子

仲間になりたい

ソニックだって丸まれば、ココみたいになれると思ったんです。多分。

スタッフコメント

気まぐれなソニックの挑戦を見守るココたち！優しい時間が流れる、ナイスなショットですね。転がって移動できるところは元々共通しているので、あとはお気に入りの植物を身につければ、立派なココの一員になれるかもしれません♪

Sephy

小さなドラゴンの冒険

ソニックの冒険で、まさかこんなに小さなドラゴンに出会うなんて！この子たちはもしかして、魔法のドラゴンの赤ちゃん？その可愛さに思わず微笑んでしまう一枚です。

スタッフコメント

『ソニックアドベンチャー2』のチャオガーデンでの1枚！まるで保護者のようにちびっこ2人を見守るソニックと、思い思いの方向に興味を示すちびっこたち。ほのぼのとした光景に、Sephyさんと同じく思わず微笑んでしまいました。

王張アルト

。＜ヘーイ、こっちこっち！！

ソニック・ザ・ヘッジホッグ06(新ソニ)の、靴にジェムを嵌めて色んな能力を使えるという設定が好きでした。うっかり一般人に踏み潰されそうなミニソニックさん。ソレアナではチリドッグを出しっぱなしにしているといつの間にかなくなっているという...

スタッフコメント

小さいソニックに呼ばれてビクッとした様子のソレアナ国民さんがいい味出してますね！確かに、自分の足と同じくらいのサイズの青い何かがものすごいスピードで走り、声までかけてきたら...平静ではいられないかも！？

TWINS STAR/歌い手

小さくなった気分！

ラグーンシティACTエミーのステージには大きい果物がたくさんあるのでなんだか小さくなった気分です！

スタッフコメント

エミーが不思議の国に迷い込んでしまったような1枚♪...というのに、実はエミーは「小さく見えるだけ」というのが面白いですね。トロピカルな世界観と仕掛けがいっぱいのラグーンシティに、エミーと一緒に遊びに行けたら楽しそう！

ロエン

あれは...！チリドッグ！！

ソニックスーパースターズのバトルモードレゴステージにて「レゴ チリドッグ」を発見...！！ 小さくてさりげなく置かれてる、オレでなきゃ見逃しちゃうね

スタッフコメント

背景におお！と思わせる物を見つけた時の高揚感、たまりませんよね～！青空の下パラソルとテーブルを出し、チリドッグを頬張るのもまた、たまらないのでしょうが...メタルファイターでは食べられなくて、歯がゆい思いをしているかも！？
