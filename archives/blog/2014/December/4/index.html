●東京ゲームショウ2014レポート！

みなさんこんにちは！

本日は、現在開催されている「東京ゲームショウ2014」セガブースからのレポートをお届け



まずは、『龍が如く0 誓いの場所』こちらは特別シアターによる映像出展1988年の神室町を舞台に金、女、暴力、本能むき出しの時代に脈動する、桐生一馬と真島吾朗の物語を描いた、『龍が如く』最新作です。



『ファンタシースター ノヴァ』は、セガ×トライエースによるストーリー性の高い本格派RPG作品
オンラインRPG『ファンタシースターオンライン2』をベースに、「オフラインで楽しめる」「アドホックプレイに特化した協力プレイ」に焦点を絞り、PlayStation®Vitaで新しいファンタシースターを創り出すために制作されたシリーズ最新作です。



国民的＆世界的に人気のアクションパズルゲーム「ぷよぷよ」と「テトリス」による夢のコラボレーションが嬉しい『ぷよぷよテトリス』。すでに発売されているPlayStation®3/PlayStation®Vita/Wii U/ニンテンドー3DSに加えて、12月4日にPlayStation®4とXbox Oneでも発売予定


アニメ、コミック、アーケードゲーム、ホビーなど、幅広いメディアに展開している男児向け新規コンテンツ『ヒーローバンク』。その家庭用ゲーム最新作『ヒーローバンク２』9/20、9/21の試遊はファミリーブースでの出展ですので、そちらをチェックしてみてくださいね。


『初音ミク Project mirai でらっくす』は、2014年7月に欧米・アジア地域向けタイトルとして発表した『初音ミク Project mirai 2』のインターナショナル版にしてシリーズ最新作が、多数のご要望を受け国内発売決定


『シャイニング・レゾナンス』は映像出展
「シャイニング」シリーズ完全新作が、PlayStation®3に初登場！本作ではバトルシステムを従来のシリーズから一新し、シンプルな操作でライブ感抜群のリアルタイムアクションバトルが楽しめる。仲間たちと力を合わせ、魔法の音色でドラゴンの力を極限まで引き出そう


完全新作のオンラインRPG『ワールド エンド エクリプス』かつて竜同士が争い、滅びた世界。大地から追われた人は、朽ちた竜の背に街を築き、日々襲い来る魔物や人同士の争いを余儀なくされていた。
今日を生き抜くために護り、明日のために戦い続ける物語を通じて、あなたはこの世界の成り立ちを目の当たりにする−。



そして、「ソニック」シリーズの新ブランドとして誕生した「ソニックトゥーン」は、Wii U『ソニックトゥーン 太古の秘宝』ニンテンドー3DS『ソニックトゥーン アイランドアドベンチャー』両タイトルが出展！試遊された方には「ソニックトゥーン」オリジナルシールをプレゼント
タイミングがあえば、ソニックにも会えるかも


他にも、セガパートナータイトルも多数出展されていますので、ぜひチェックしてくださいね
