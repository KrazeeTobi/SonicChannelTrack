セガブースのセガスタジオにてイベント開催、WEBにてライブ配信！
セガブースでは今年も盛りだくさんのイベントをご用意しております。期待の新作のプレゼンテーションはもちろん、豪華ゲストをお招きした番組も予定しているので、どうぞご期待ください。また、イベント開催中はその様子をWEB上にてライブ配信（ニコニコ動画生配信）いたします。一般公開日のスタジオイベントだけでなく、一般のお客様が入場できないビジネスデイのイベントの様子もリアルタイムでご覧いただけますので、直接イベントを見にこられない方はぜひ、配信をチェックしてみてください。
※一部ライブ配信を行わないステージ、アーカイブ配信を行わないステージもございます。予めご了承下さい。
