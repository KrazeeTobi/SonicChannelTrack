Lv上げ効率アップレポート
-------「準備編」-------





■　　レベル80になることを目標とします。
　　　その為に最も効率のいいレベル上げを模索します。

■　　マグをすでに一つ完成しているとします。
　　　このマグの能力値については解釈自由です。
　　　大体の目安として、以下の能力値を想定しています。
　　　ーーーーーーーーーーーーーーーーー
　　　ハンター　　POW 180 / DEX 0 / MIND 0
　　　レンジャー　POW 120 / DEX 70 / MIND 0
　　　フォース　　POW 0 / DEX 0 / MIND 180
　　　ーーーーーーーーーーーーーーーーー

■　　フォースキャラクタの場合は、
　　　ある程度の攻撃・補助テクニックを
　　　覚えた状態を想定しています。
　　　目安レベルとして、
　　　ーーーーーーーーーーーーーーーーー
　　　　　　ノーマル＝１５レベル
　　　　　　　ハード＝２０レベル
　　　　ベリーハード＝２５レベル
　　　アルティメット＝２５レベル以上
　　　ーーーーーーーーーーーーーーーーー
　　　と仮定します。
　　　中・上級テクニックはそれより４〜５レベルほど低くても
　　　Lvアップに大きな問題はありません。

　　　ただし、テクニックの強さによっては
　　　乱入プレイ時に瞬殺キャラクタになってしまう為
　　　オンラインでの攻撃テクニックはある程度自粛しましょう。

■　　育成の基本条件として、以下を挙げます。
　　　このポイントから外れるものは
　　　いっさい掲載していませんし、紹介しません。


　　　・マテリアル使用を考慮しない（一部例外とする）

Lv上げ効率アップレポート
-------「基本編」-------



「基本編」に示したマグを使用して、
オフラインEP1だけをメインに、
クエスト等も無視すれば、
ベリーハード遺跡をクリアするまでに
Lv41程度にはなっているはずです。

基本的に、ハンターならセイバーやパルチザン、マシンガン。
レンジャーならショットとハンドガンを駆使。
フォースはテクニックオンリーで進めていきます。

Lv41から上を目指すのが、職業ごとに個性が出てきます。
VHやUltを周回をこなしながら、EP2エリア埋めしながら、
またはクエストをクリアしながらでも
割と余裕で行けるはずですが、
腕に自信があるなら、フォースならばLv50〜60で
アルティメットも押し進められます。
効率がいいのはアルティメット坑道だそうです。
それなりにレアアイテムも揃うでしょう。
ハンター・レンジャーでも、
Lv60くらいからUlt森以上を周回するようにすると
より高い効率を得られます。







＝各難易度別ダークファルス攻略＝


[ ノーマル編 ]

　最低必要HP　131
　必要光耐性　0
　　ノーマルは第３形態がないので楽。
　　ハンターならマシンガン連射、レンジャーは高属性単発銃。
　　アンドロイドなら、適度にドロー系エレメントで
　　HPを吸収回復しつつ攻撃。
　　フォースは何も考えず第１形態ではゾンデ、
　　第２形態ではバータを連射。
　　ラバータ》ラフォイエコンボに備えて、
　　スケープドールは３個は持っていきたい…。


[ ハード編 ]
　最低必要HP　251
　必要光耐性　34
　　第３形態が鬼門。光耐性をソリッドフレームや
　　レジスト/ホーリーx2等で上げて確実に望む。
　　アンドロイドの場合はあらかじめディメイト等を
　　床に20〜40個程置いてから(1box辺り5個が適正)挑む。
　　やはりアンドロイドはドロー系で攻撃。
　　フォースはフォイエで。
　　ハンター・レンジャー(特にアンドロイド)は速攻勝負。
　　できれば単発銃とマシンガンを１丁ずつ持ち込むのが吉。


[ ベリーハード編 ]
　最低必要HP　391
　必要光耐性　16
　　第２形態を如何に乗り切るかが問題。
　　吸収には要注意。
　　グランツに耐えられれば第３形態は楽かも知れない。
　　Lv40で挑むのは非常に危険ということも
　　頭に入れておいてください。
　　長引くとこちらが完全に不利です。
　　チョップ攻撃とHP調節によっては
　　マグ無敵発動を容易にできます。



その他ボスの攻略法、
Ultでの攻略法もお待ちしておりますm(_ _)m

＝小技・大技・意味ナシ技のコーナー＝


「ラゾンデよりジェルンザルア！」
　　DC版では敵出現と同時にラゾンデを撃つ事で
　　経験値80%を確保するのが常套手段とされていましたが
　　今回のGC版でのラゾンデは範囲が狭くなっている為、
　　ジェルン・ザルアを使用する事でも
　　同様の効果が得やすくなっています。
　　今回よりジェルン・ザルアは、
　　ラゾンデ型の効果範囲に変更されているのがその理由。
　　さらに、補助効果はより高いLvを優先とする為、
　　より上のLvのテクニックでかけ直したりする必要もなく、
　　遠慮なく弱体化補助を仕掛けて行けます。
　　もちろん、補助Lvが低くて無効化されたとしても
　　ちゃんと80%の経験値は入ります。
　　DC版に慣れた方には、乱入先でその旨を
　　細かく説明してあげた方が、
　　お互い気持ちいい冒険ができるかも知れません（＾＾；
　　　===レポート提出：てれすさん



「ダークファルスの誘導弾でグランツ避け!?」
　　ダークファルスの厄介な攻撃の一つ、グランツ。
　　これを簡単に避ける方法があります！
　　ダークファルスが誰にも乗り移っていない状態の
　　上空攻撃に「誘導弾」がありますが、
　　これの２セット弾４発の内、
　　最後の１セットにわざと当たります。
　　そして倒れたら起き上がりと同時に回復。
　　ダークファルスのグランツも来ますが、
　　起き上がり無敵でやり過ごせます。
　　この方法の欠点としては、
　　チョップ攻撃を凌ぎ切れない場合は
　　どうしてもスケープドールが多く必要になってしまう事。
　　アルティメットの誘導弾ではタイミングが取り辛い事。
　　オンラインでは狙い辛い作戦であるという事。
　　この３点でしょうか。
　　低レベルクリアを目指すなら必須のテクニックです。
　　　参考までに…誘導弾攻撃の威力（オフライン）
　　　　ハード　　　　＝130
　　　　ベリーハード　＝370
　　　　アルティメット＝720
　　　===レポート提出：ゆんへーさん



「ダークファルスのグランツでチョップ避け!?」
　　ハード、VHのダークファルスでは、
　　とことん光耐性を上げてグランツだけ耐えられれば
　　スケープドール持ち込みでもなんとかなるようです。
　　VHでグランツに耐えられ、反面チョップ攻撃では即死
　　こんな場合はグランツを食らって倒れ、
　　チョップ攻撃の１発目を起きあがり無敵でやりすごす事が
　　可能なようです。
　　もちろん、起き上がり無敵時間内に
　　レスタ・メイトやドロー系EXなどでの回復も忘れずに。
　　１発目チョップ以外は当たりにくいらしいです。
　　しかし、乗り移り解除直後のチョップだけは回避不能です。
