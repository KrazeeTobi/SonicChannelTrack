真・廃人度チェック!!?(↓レス追加！)



ということで、数値的にもっと詳しく？調べるマッスィーン（？）
個人的に考えた物です、計算結果の求め方について
意見のある方はお願いします（ぺこり




オンオフ問わず、潜って遊ぶ時間の割合：チャットの割合

を、まず記述してもらって…
それに持ちキャラのLvを別途、計算。




例）
ゲーム７０：チャット３０
Lv108 + 89 + 51 + 55 + 52 + 51 + 35 + 3 = 合計Lv444
8キャラ




んで、444（キャラ合計Lv）をゲーム割合値で割る。
　444÷70=Ｘ
　　　 6,3=Ｘ

そのＸ値をゲーム・チャットそれぞれの割合値でかけてみる。
　6,3×70=441=ゲーム時間指数
　6,3×30=189=チャット時間指数

んで、割合値を直してみる。

ゲーム時間指数441 : チャット時間指数189

そして、チャット時間指数は
廃人度に直接響く要素ではないだろう、という事で
最終計算時に修正。

さぁ、最終計算。これで廃人度を求めます。

ゲーム時間指数+(チャット時間指数x0.5)+(キャラ数x10)＝廃人度
（秘かにキャラ数による計算が入ってます）



おお？どうだろうかな。
ちなみに例題で言えば

441 + ( 189 x 0.5 ) + ( 8 x 10 ) = 廃人度
441 + (　　94　　) + (　　80　　) =廃人度
　　　　　　　　　　　　　　　　615=廃人度

ででで、出たァーーーーーッ！
例で言うと、615が、廃人度。
しかし、これの難しい所は
計算がややこしくて、さらに、多くのサンプルがないと
正しい称号ランクが考えられない罠。
でも、より正確な値が出るとは思うんですけどね〜

あ、購入した日から起算して、
何日でこの状況になったか、とかも
計算結果に入れてみると面白いかも。
あ、それは日が浅ければ浅いで「多くやりすぎ」、
日が経っていれば「長くやりすぎ」になるので
別にいいかぁ（＾＾；
