[長文注意]CATVでPSOをする場合の注意事項

＃　情報が劣化している場合があります
＃　詳細は、各自確認のこと

まず、質問するならケーブルテレビ局名を書きましょう
ケーブルテレビ局によって、状況は千差万別です
局名がわからないなら「郵便番号(上３桁)」でも構いません

＃個人的には「CATVで繋いでます」は「電話回線で繋いでます」と等価

PSOをオンラインモードで遊ぶには、普通はBBAが必須です
# 普通じゃない方法は割愛

CATVでケーブルモデム直結の場合、基本はパソコンと同じです

・DHCPで接続するように指定されているならDHCP
・特定のIPアドレスが割り当てられているなら、それを設定
・DNSアドレスが指定されているならそれを設定
という風に設定します

DHCPで設定する多くの局は何も必要ありませんが、例えば、
J-COM@NetHomeはDHCPホスト名(=コンピュータ名)が必要なはずです
# 私はJ-COM@NetHome使ってませんので実際どうかは知りません
# 最近は、コンピュータ名が必要ない方向に移行しているそうです
# 従って、局によってはDHCPホスト名は必要ないでしょう
# 旧タイタス系の局で駄目な場合、一度、モデムの電源を切りましょう

DHCPでIPアドレスが設定されるケーブル局の場合は、
PCで使っていると、多くの場合IPアドレスを解放する必要があります
# IP解放＝局から貸し出されているIPアドレスを局に返還する

＜PCのIPを解放(リリース)する方法＞

Win系：(DOS)コマンドプロンプトで「ipconfig /release_all」
# Windows95 にはありません
Win9X系なら「winipcfg」(GUI)で「(全て)解放」でもOK

MacOSXはターミナルでsuして「dhclient -r」かなぁ

MacOS9系は
「コントロールパネル」「ＴＣＰ／ＩＰ」で
「Ｅｔｈｅｒｎｅｔを違うものに変換する」
# 「ＰＰＰ」とか？
最後に保存すること

＜／PCのIPを解放(リリース)する方法＞

この操作が必要かどうかも含めて、自分の局で確認しましょう

繋ぐ機器を変更すると、特別な操作・手続きが必要な場合があります
例えば、モデムのリセット(数分間電源断の場合も)
MACアドレスの登録、ケーブル局への届け出等です
複数台同時に接続する場合は、手続きが必要な場合があります

# ケーブルモデムの電源断は指示されない限り行わない方が無難です
# 頻繁に電源を操作すると、ケーブルモデムを傷める恐れがあります
# ケーブルモデムの電源断は必要性を確認してから行いましょう

尚、固定でIPアドレス等が指定されている場合はそれに従って下さい
※注　同じIPアドレスの機器(PCとGC等)は同時に接続できません

詳しくは、ケーブル局に問い合わせましょう

GC版PSOのページのサポートに一部ですが動作確認情報が出ています
http://www.sonicteam.com/psogc/pc/manual_net.html

参考に、DC版BbAの実績なら、DC版BbAの販売サイトにありました
が、2003年3月末でサイトクローズしました
旧BBABBSはまだ生きているので、ここで探してみるのも良いでしょう
http://e-crm.csi.co.jp/dc/dc_bbslog.nsf/MainView

・UDPに関して

グローバルIPアドレスであれば、基本的にUDP ONでも大丈夫です
あまり有効に使えてない節がありますが・・・

ルータを使用する場合は、ルータを設定しない場合は初期値のままで

尚、
基本的にプライベートIPアドレスが割り当てられるケーブル局の場合は
ルータを設定しても初期値「UDP自動(=OFF)」のままです

使用する場合は自己責任に於いて使用して下さい

実は、BBAの設定が初期状態のままでも繋がるかも知れませんが・・・

１．環境設定書(?)を手元に準備

２．設定書を参考にＧＣを設定(=在ればDHCPホスト名を入力)

（パソコンを繋いでいた場合は、パソコンでIP解放）

３．ケーブルモデムとGCを接続

ダメだった場合、旧タイタス系の場合はモデムの電源断

ルータがある場合は、BBA買ってきて繋ぐだけでOKでしょう
