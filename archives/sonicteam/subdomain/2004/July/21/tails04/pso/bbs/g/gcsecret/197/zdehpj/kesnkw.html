私は昨日から久々のマグ育成を始めました。
私のマグ作成レシピが途中まで参考になるかもしれないので
私のレシピをコピペします。

レンジャーBグループで LV50 の時点で DEX > POW > MID となるようにマグに餌をやる。

アンティドートx10 DEF 05 + 50 POW 01 + 00 DEX 04 + 00 MID 00 + 00 = LV10

これで LV10 となりカルキ(エストラ)になる。

アンティドートx100 DEF 05 + 50 POW 06 + 00 DEX 19 + 00 MID 00 + 00 = LV30

アンティドートx20 DEF 05 + 50 POW 07 + 00 DEX 22 + 00 MID 00 + 00 = LV34

アンティドートx7 DEF 05 + 50 POW 07 + 35 DEX 23 + 05 MID 00 + 00 = LV35

これで LV35ミトラ(バイラ)になる。

先に半端になってるPOWをモノメイトで1つ繰り上げる。

モノメイトx22 DEF 05 + 50 POW 08 + 01 DEX 23 + 05 MID 00 + 00 = LV36

アンティパラライズx70 DEF 05 + 00 POW 08 + 00 DEX 37 + 05 MID 00 + 00 = LV50

これで LV50カイタバ(マイラ＆ユウラ)になる。

アンティドートx100 DEF 05 + 00 POW 08 + 00 DEX 53 + 05 MID 00 + 00 = LV66

アンティドートx13 DEF 05 + 00 POW 08 + 00 DEX 55 + 13 MID 00 + 00 = LV68

モノメイトx466 DEF 05 + 00 POW 49 + 94 DEX 55 + 00 MID 00 + 00 = LV109

(もしくは ディメイトx381 DEF 05 + 00 POW 49 + 91 DEX 55 + 13 MID 00 + 00 = LV109)

SKYLY or PINKAL or YELLOWBOZE のフォース♂に渡した後、モノメイト1個与えてニドラーにする。

モノメイトx1 DEF 05 + 00 POW 50 + 03 DEX 55 + 00 MID 00 + 00 = LV110

これで LV110ニドラーになる。
