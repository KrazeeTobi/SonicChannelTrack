Lv上げ効率アップレポート
-------「準備編」-------





■　　レベル80になることを目標とします。
　　　その為に最も効率のいいレベル上げを模索します。

■　　マグをすでに一つ完成しているとします。
　　　このマグの能力値については解釈自由です。
　　　大体の目安として、以下の能力値を想定しています。
　　　ーーーーーーーーーーーーーーーーー
　　　ハンター　　POW 180 / DEX 0 / MIND 0
　　　レンジャー　POW 120 / DEX 70 / MIND 0
　　　フォース　　POW 0 / DEX 0 / MIND 180
　　　ーーーーーーーーーーーーーーーーー

■　　フォースキャラクタの場合は、
　　　ある程度の攻撃・補助テクニックを
　　　覚えた状態を想定しています。
　　　目安レベルとして、
　　　ーーーーーーーーーーーーーーーーー
　　　　　　ノーマル＝１５レベル
　　　　　　　ハード＝２０レベル
　　　　ベリーハード＝２５レベル
　　　アルティメット＝２５レベル以上
　　　ーーーーーーーーーーーーーーーーー
　　　と仮定します。
　　　中・上級テクニックはそれより４〜５レベルほど低くても
　　　Lvアップに大きな問題はありません。

　　　ただし、テクニックの強さによっては
　　　乱入プレイ時に瞬殺キャラクタになってしまう為
　　　オンラインでの攻撃テクニックはある程度自粛しましょう。

■　　育成の基本条件として、以下を挙げます。
　　　このポイントから外れるものは
　　　いっさい掲載していませんし、紹介しません。


　　　・マテリアル使用を考慮しない（一部例外とする）
　　　　　マテリアル不使用時でも十分な効率を得られる事を
　　　　　前提として紹介していますが、
　　　　　一部キャラクタにおいてＨＰマテリアルを使用した
　　　　　Lv上げ効率アップレポートを紹介しています

　　　・敵の倒しやすいオフラインをメインに捉える
　　　　　効率次第でオンラインも視野に
　　　　　オンラインを中心としたLv上げも
　　　　　いつか欄外で紹介するかもですが…（＾＾；

　　　・全てのクエストプレイ制限なし
　　　　　クエスト中断キャンセルも手段とします

　　　・不正は全面禁止
　　　　　オンライン協力プレイにおいても禁止

　　　・EX/オートツールレベル上げは対象外
　　　　　ここでは取り上げません

　　　・極端にレア度の高い武器は使用しない
　　　　　誰でも出来る、がポイントと考えます

　　　・レアアイテム効率は「出ればいいな」程度
　　　　　メインはレベル上げ、を念頭に…



■　　育成をはじめる前に移動することを推奨するアイテム。
　　　手持ちで賄えない場合、友人から借りるのも手？
　　　…Lv上げ中に出たモノを代わりにプレゼントとかなら
　　　アリかも知れませんね（＾＾；
