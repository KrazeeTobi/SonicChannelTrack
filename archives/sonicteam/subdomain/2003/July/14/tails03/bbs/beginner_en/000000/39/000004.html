YOUR DISC IS FINE! The server wasn't ready until June 23rd. The download page wasn't ready until about last week. The BBS wasn't open until days ago. If you try again you can get the "Sonic Theme" which replaces the menu-woman's voice for Sonic's. But if you're getting a message that says "Access Forbidden" while using the Dreamcast then the server has the problem. PC's will get that message if they try to access DC-only content (the downloads). This advice comes from knowing how to set up servers and websites. Trust me, your disc should be fine!

PS. I myself thought it was something on my end. The Dreamcast has data like localization, language, and serial number stored in it's built in memory. Some sites only allow access to specific countries or computer brands. The Sonic Adventure 1 site only allowed DC users in Europe to play in UK-only contests. A simple call to 1-800-SEGA-USA on June 20th told me that the web site was not up yet. But now it is!
