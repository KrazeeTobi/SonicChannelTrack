> My Chao Garden won't work! It says the file is damaged. It will save then when I go in the Chao World the file says failed! Can someone help me please! Is it due to a bad memory card? Someone tell me please! It also says not enough memory but my card has 177 spaces left! Help me!

You cannot have 177 blocks in a 200 block memory card and have a game loaded. You have three files, if you never go to chao garden with a new game you have only one file of 18 blocks, leaving 182 blocks. If you go in the garden and place a chao in a MC for the first time you generate 2 more files, one of 52 blocks, leaving 130 free and generate another one of 128 blocks, leaving only 2 blocks.... and if you download the ST specials of 2 blocks you have nada, zip...zero. If you have an extra MC try saving the game file first. Run the game as a new one(you can always erase if it is no good)... new chao/garden files should be generated if its ok. Hatch egg and put in MC... if it works ok then erase the 52 block and 128 block files and copy 128 block from the MC your trying to rescue(bad???). fire up the game again run the same tests. If ok then again erase 52 and 128 block and copy over both from suspect MC. If every thing works ok now you have saved some if not all your data. Erase everthing on the suspect MC, reformat and make a duplicate copy of everthing you saved on the new MC. If it works ok... the memory card was glitzed for some reason... shutting off when still saving... removing game MC and then playing game before reinserting... going into Health Center(stay away)...etc.
