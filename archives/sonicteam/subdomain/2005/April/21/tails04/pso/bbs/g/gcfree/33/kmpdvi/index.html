・私的簡易マニュアル

※タグを表示させる為に一部無理に全角文字を使用しています。

【閲覧の仕方】

　BBSの要、閲覧についての説明です。


　まず、BBSを開いた時にページ上部及び下部に

　　　New ← ○ ○ ・・・・・ ○ ○ → Old

という感じのものが目に付くかと思います(○は数字)。この数字の一番左の番号
が現在の最新のページです。そこから右に行くほど古いページとなります。この
数字の部分を選択することによりページを切り替えることが可能です。過去ログ
を参照する時等に活用してください。


　次に各発言の参照ですが、パターンが色々あるので例を一つあげてみます。

　▼ 親発言 親発言者 yyyy/mm/dd hh:nn:ss
　　├返信A 返信者A yyyy/mm/dd hh:nn:ss
　　│└親返信 親発言者 yyyy/mm/dd hh:nn:ss (修正1回)
　　├返信B 返信者B yyyy/mm/dd hh:nn:ss
　　├DELETED 返信者D yyyy/mm/dd hh:nn:ss (削除)
　　└返信C 返信者C yyyy/mm/dd hh:nn:ss

　上記の場合、親発言に対し返信者Aさん・返信者Bさん・返信者Cさん・返信者Dさん
が返事を書いていて返信者Aさんに対してさらに親発言者さんが返事を書いています。
このうち、返信者Dさんの発言は既に本人により削除されています。また、親返信に
関しては一度修正がされていることをあらわしています。

　各発言を個々に参照したい場合、参照する発言のタイトル
(この場合、「親発言」「返信A」等)を選択します。そうでなく発言に対する
ツリー全体を参照したい場合は、親発言の左にある「▼」を選択します。



【発言・返信】

　新しく発言したり、既にある発言に対して返事を書く為の説明です。

　新しく発言する時は同様の発言が無いか過去ログを見て確認しましょう。
検索機能が無い為全て見てとまでは言いませんが、せめて２・３ページ出来れば５ページ
くらいはさかのぼって調べてみてください。
　また、セガの規約等も十分認識し違反しないように注意してください。


　新規発言をする場合はページ右上にある「新しく発言する」を選択し、
MAKING BASE NOTE ページを開きます。必要項目及びタイトル・本文を入力後WRITEボタンを
押下することで新規発言できます。

　既にある発言に対し、返事を書きたい場合は対象の発言を表示(単体でも一覧でも可)
し、返事を書きたい発言にある「返事を書く」を選択し、MAKING APPEND NOTE ページを
開きます。必要項目及び本文を入力後APPENDボタンを押下することで、対象の発言に対して
返事が書けます。
　返信に関しては返事の内容の一行目がタイトルとして使用されます。ですので、いきなり
本文を書き出すのではなく、タイトル文を書いて１・２行改行してから書き込むと良いかと
思います。


　各書き込み時、「以後ステータス情報をブラウザに保存する(cookie) 」にチェックを
入れておくことで、メールアドレスと確認キーについて保存することが可能です。少しでも
入力の手間を減らす為に活用してください。

　メッセージは色を付けたり文字の大きさを変えたりすることが可能です。メッセージの
中に直接HTMLのタグを使用することで実現が可能です。

　　例）赤色にする　→　＜font color=red＞赤色＜/font＞にする
　　　　大きくする　→　＜big＞大きく＜/big＞する

　上記はホンの例ですので、色々試したい方はHTMLタグのページ等を参照してください。
ただし、無駄に乱用したり変に色を変えると非常に見辛くなります。そういうところは
注意してください。

　使用できるタグ：＜font＞・＜h＞・＜b＞・＜u＞・＜i＞・＜big＞・＜small＞・＜center＞

　リンクに関しては文章内にURLを直接記述することで自動的にリンクが張られます。
＜A＞タグを使用する必要はありません(使用してもそのまま表示されてしまいます)。

　発言する前に書いた内容を確認しましょう。相手を不快にさせたりしないかよく確認して
から登録するようにしてください。相手や第三者に不快を与える書き込みでは書き込まない
ほうがましです。
　また、発言の内容がフリー向けかシークレット向けかよく判断してください。
特にアイテム名やクエスト等の事を書く時は注意が必要です。SWとして出るアイテム名は
例え沢山出る物だといってもシークレットに書くべき内容になります。また、ネタバレに
ならなくてもクエストの内容や回答にネタバレが含まれる可能性が予想される内容は
シークレットに書き込むべきです。

　本文は基本的に右端で自動的に折り返されますが、自動的に改行されない為本文が長すぎ
ると非常に見辛くなります。適度に改行を入れて読みやすい文章にするようにしてください。
また、入力エリアに一行ずつ空白を入れたりするのは非常に醜くするだけです。スペースでは
なく改行を使用してください。



【発言の修正・削除】

　発言した内容を修正する為の説明です。

　発言を書き込んでしまった後に、文章が間違えていたり誤字脱字が多い等、色々修正
したくなる時があると思います。そういう場合はノートメニューを活用しましょう。


　まず、自分の発言を表示(単体でも一覧でも可)し、自分の発言にある「ノートメニュー」を
選択し、NOTE MENU ページを開きます。必要項目を入力し修正したい部分を修正してREVISEボタン
を押下することで対象の発言を修正できます。この時近くにDELETEボタンもあるので間違って
押してしまわないように注意してください。
　また、返信の場合に限り、一行目を修正することでタイトルを変更することが出来ます。
親発言のタイトルを変更することは出来ないようです。

　削除する場合はノートメニューを開いて、必要項目入力後一番下にあるDELETEボタンを
押下すれば削除できます。

タグについて補足（失敗しないためにw）

Fontタグで囲う場合は、必ずはじめと終わりの表記を合わせないと失敗します。

たとえば
<Font Color="red">赤色</font>
のようにそのまま出てきてしまいます。

あとタグで括れるのは１行だけです。
複数行にまたがるのは無視されます。
<font Color="red">1行目
２行目
３行目</font>

あと書いた後には必ず自分の発言を開いて失敗していないか確認することが失敗を減らすこつです。

では。

<font color=red size=+1>
　この文章は、文字色をred（赤）に、
　文字のサイズを基準サイズより１段階大きく。
　という意味を持っています。
　当然、redをblueやyellow等にすることもできます。

　ボールド体にします。

　もちろん、書き込みの終点までってのは効果としてイマイチなので、
　修飾末尾で</b>を忘れずに。


<i>
　文字に斜体効果を持たせます。
　これも使い方はボールド体と同じです（＾＾；


ウラ技：隠しメッセージ

　HTMLとは「Hyper Text Markup Language」の各単語の頭をとったもので、
テキストにマークを付けてテキストを越えさせる為の言語とでもなるでしょうか。

出きるわけです。表示には対応したソフト(この場合ブラウザ)が必用なのですけどね。

　ネット上では検索で「HTML タグ」の二つで検索すれば、うわっと言うくらい
沢山ヒットすると思います。書店でもホームページ作成の為のコーナー(?)に

・Q&Aバージョン

【閲覧の仕方】
Q1-1.
　ページを変えるにはどうすれば良いですか？
A1-1.
　まず、BBSを開いた時にページ上部及び下部に

　　　New ← ○ ○ ・・・・・ ○ ○ → Old

という感じのものが目に付くかと思います(○は数字)。この数字の部分を選択
することによりページを切り替えることが可能です。


Q1-2.
　一番Old側のページを表示してもそれ以前のページが出てこないのですが。
A1-2.
　BBSで保存されるページ数は２０ページまでのようです。通常ではそれ以前の
ページを見ることは出来ません。PC等を利用すればURLを直接指定する事により
より過去のページが見れる可能性があります。


Q1-3.
　一つ一つ発言見ていくのは面倒なのでツリーの内容を一度に表示できませんか？
A1-3.
　親発言のタイトルの左に▼マークがあるかと思います。それを選択することで
一括表示が可能です。


Q1-4.
　ツリー内のある発言以下を一括表示出来ませんか？
A1-4.
　一括表示は▼によるもののみです。ですので、出来ません。


Q1-5.
　さっきまであった書き込みが無くなっているのですが。
A1-5.
　恐らくは、発言内容に不備が合ったため管理者に削除されたのでしょう。
発言されたツリーが削除されてしまった場合、もう一度書き込んだ内容に
問題が無かったか思い返してみてください。





【発言・返信】
Q2-1.
　質問したい内容が見つからないので新規に質問したいのですが。
A2-1.
　ページ右上にある「新しく発言する」を選択し、MAKING BASE NOTE ページを
開いてください。発言に必要な項目を入力し、WRITEボタンを押下することで
新規に発言できます。この時、一度自分が回答する身になってみて足りない情報
が無いかよく確認してください。情報をキチンと書き出しておけば優しい方が
返信してくれると思いますよ。
　出来れば、タイトルを見ただけで何を聞きたいかが分かるようにした方が良いです。


Q2-2.
　分かる質問があったので返事を書いてあげたいのですが。
A2-2.
　返事を書きたい発言にある「返事を書く」を選択し、MAKING APPEND NOTE
ページを開いてください。発言に必要な内容を入力してAPPENDボタンを押下する
ことで返事を付けられます。この時、本文の一行目がタイトルとして使用される
ので、出来るだけ分かりやすいタイトルを付けてあげた方が良いです。


Q2-3.
　新規に書き込もうと思うけどフリーとシークレットどっちに書けばいいかな？
A2-3.
　まず、固有グラフィックを持つアイテムや仕様に関連する内容及びクエストの
登場人物や内容に関しての事であれば、シークレットで書き込むべきです。そうで
なく、接続の方法やシステムに関してもマニュアルに載っているようなものに関し
てはフリーで良いかと思います。


Q2-4.
　これってバグと思うんだけど、シークレットにだったら書き込んでいいんだよね。
A2-4.
　バグと思うなら一切書き込まずにソニチへその内容をメールしてください。もし、
本当にバグとしても、その内容が不特定多数に知れ渡りオンのバランスを崩す恐れが
あります。繰り返しますが、バグと思うなら一切書き込まないでください。


Q2-5.
　返事を書いているうちに「このノートはこれ以上発言できません。」とでたのですが。
A2-5.
　一つのツリーに発言できる個数が決められております。ですので、それに達している
為のメッセージです。ツリーが大きくなる場合は、お礼のレスは一括もしくは親発言を
修正して一括で書き込む方が良いです。





【発言の修正・削除】
Q3-1.
　書き込んだ後に間違いを見つけたんだけど、修正できないのかな？
A3-1.
　まず、自分の発言を表示(単体でも一覧でも可)し、自分の発言にある「ノートメニュー」を
選択し、NOTE MENU ページを開いてください。必要項目を入力し修正したい部分を修正して
REVISEボタンを押下することで対象の発言を修正できます。この時近くにDELETEボタンもある
ので間違って押してしまわないように注意してください。


Q3-2.
　タイトルに誤字あったんだけどなおせないのかな？
A3-2.
　親発言のタイトルの場合、残念ながら修正は不可能みたいです。そうでなく、返信の場合は
修正にて一行目を修正することで可能です。


Q3-3.
　発言内容が合わないから消してといわれたのですが。
A3-3.
　まず、自分の発言を表示(単体でも一覧でも可)し、自分の発言にある「ノートメニュー」を
選択し、NOTE MENU ページを開いてください。必要項目を入力しDELETEボタンを押下すること
で削除することが出来ます。
