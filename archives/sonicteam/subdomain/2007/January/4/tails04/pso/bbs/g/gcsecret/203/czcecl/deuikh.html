エネミー編

こちらはエリア順にお送りします。

色で判別しやすいエネミーの名前にはそのエネミーの色がついています。

＝＝＝＝＝＝＝森＝＝＝＝＝＝＝


＜ブーマ＞
（Booma）
＊プーマ（「フ」に半濁点)
＊ゴブーマ、ジゴブーマについても同じ。
＊弱 ブーマ＜ゴブーマ＜ジゴブーマ 強


＜バートル＞
（Bartle）
＊パートル（「ハ」に半濁点）
＊元ブーマです。


＜バーブル＞
（Barble)
＊パーブル（「ハ」に半濁点）/ バープル（「フ」に半濁点）/ パープル（「ハ」「フ」に半濁点）
＊元ゴブーマです。


＜ラグ・ラッピー / アル・ラッピー＞
（Rag Rappy / Al rappy）
＊難易度はノーマル、ハード、ベリーハードです。
＊希少種は青色のアル・ラッピーです。


＜エル・ラッピー / パル・ラッピー＞
（El Rappy / Pal Rappy）
＊難易度はアルティメットです。
＊バル・ラッピー（「ハ」に濁点）
＊エル・ラッピーの希少種がパル・ラッピーです。


＜サベージウルフ　/ バーベラスウルフ＞
（Savage Wolf / Barbarous Wolf）
＊サベージウルフが子分で緑っぽい色です。
＊バーベラスウルフが親分で青っぽい色です。


＝＝＝＝＝＝＝洞窟＝＝＝＝＝＝＝


＜パルシャーク＞
（Pal Shark）
＊バルシャーク（「ハ」に濁点）
＊弱 エビルシャーク＜パルシャーク＜ギルシャーク 強


＜バルマー＞
（Vulmer）
＊パルマー（「ハ」に半濁点）
＊元エビルシャーク


＜ゴバルマ＞
（Govulmer）
＊ゴパルマ（「ハ」に半濁点）
＊元パルシャーク


＜ポイゾナスリリー / ナルリリー＞
（Poison Lily / Nar Lily）
＊難易度はノーマル、ハード、ベリーハードです。
＊ポイゾナリリー
＊希少種は赤色のナルリリーです。


＜オブリリー / ミルリリー＞
（Ob Lily / Mil Lily）　
＊難易度はアルティメットです。
＊オプリリー（「フ」に半濁点）
＊希少種は青色のミルリリーです。


＜グラスアサッシン / クリムゾンアサシン＞
（Grass Assassin / Crimson Assassin）
＊グラスアサッシンが難易度ノーマル、ハード、ベリーハードです。
＊クリムゾンアサシンが難易度アルティメットです。


＜ナノノドラゴ＞
（Nano Dragon）
＊ナノノドラゴン
＊ナノノドラコ


＜プフィスライム / プイィスライム＞
（Pofuilly Slime / Pouilly Slime）
＊ブフィスライム（「フ」に濁点）
＊ブイィスライム（「フ」に濁点）
＊希少種は赤色のプイィスライムです。


＜パンアームズ＞
（Pan Arms）
＊バンアームズ（「ハ」に濁点）


＜ミギウム / ヒドゥーム＞
（Migium / Hidoom）
＊ミギウムが青色です。
＊ヒドゥームが肌色です


＜デ・ロル・レ / ダル・ラ・リー＞
（De Rol Le / Dal Ra Lie）
＊デ・ロル・レが難易度ノーマル、ハード、ベリーハードです。
＊ダル・ラ・リーが難易度アルティメットです。


＝＝＝＝＝＝＝坑道＝＝＝＝＝＝＝


＜ギルチック / ギルチッチ＞
（Gillchic / Gillchich）
＊ギルチックが難易度ノーマル、ハード、ベリーハードです。
＊ギルチッチが難易度アルティメットです。


＜ダブチック / ダブチッチ＞
（Dubchic / Dubchich）
＊ダプチック / ダプチッチ（「フ」に半濁点）
＊タブチック / タブチッチ（「タ」に濁点無し）
＊ダブチックが難易度ノーマル、ハード、ベリーハードです。
＊ダブチッチが難易度アルティメットです。


＜ギャランゾ / バランゾ＞
（Garanz / BARANZ）何故かバランゾは半角大文字に変換されました（笑
＊ギャランゾが難易度ノーマル、ハード、ベリーハードです。
＊バランゾが難易度アルティメットです。


＜ボルオプト[ver2]＞
（Vol Opt[ ver.2]）
＊ポルオプト（「ホ」に半濁点）
＊ボルオブト（「フ」に濁点）
＊ポルオブト（「ホ」に半濁点、「フ」に濁点）
＊ver2を英語に変換するとrと2の間に.がつきます。


＝＝＝＝＝＝＝遺跡＝＝＝＝＝＝＝


＜ビーエル / ジーエル＞
（Bee L / Gee L）
＊ビーエルが難易度ノーマル、ハード、ベリーハードです。
＊ジーエルが難易度アルティメットです。
＊供にソーサラーから見て右側にいます。
＊ビーエルはEP1だとラフォイエ、ギバータを、EP2だとギゾンデ、ギバータを撃ってきます。
＊ジーエルはグランツ、メギドを撃ってきます。


＜ビーアル / ジーアル＞
（Bee R / Gee R）
＊ビーアルが難易度ノーマル、ハード、ベリーハードです。
＊ジーアルが難易度アルティメットです。
＊供にソーサラーから見て左側にいます。
＊供にレスタを唱えます。


＜カオスブリンガー / ダークブリンガー＞
（Chaos Bringer / Dark Bringer）
＊カオスブリンガーが難易度ノーマル、ハード、ベリーハードです。
＊ダークブリンガーが難易度アルティメットです。
＊ブリンガーライフルはカオスブリンガーが１発、ダークブリンガーが３発撃ってきます。


＝＝＝＝＝＝＝神殿＝＝＝＝＝＝＝


＜バルバレイ＞
（Barba Ray）
＊パルバレイ（１文字目の「ハ」に半濁点）
＊バルパレイ（３文字目「ハ」に半濁点）
＊パルパレイ（両方の「ハ」に半濁点）


＝＝＝＝＝＝＝宇宙船＝＝＝＝＝＝＝


＜ゴル　ドラゴン＞
（Gol Dragon）
＊ガル　ドラゴン


＝＝＝＝＝＝＝中央管理区＝＝＝＝＝＝＝


＜メリルリア / メリルタス＞
（Merillia / Meriltas）
＊メリルリアが雌で赤色です。
＊メリルタスが雄で緑〜青っぽい色です。


＜メリカロル / メリクル / メリキュス＞
（Mericarol / Merikle / Mericus）
＊メリカロルが赤色で難易度ノーマル、ハード、ベリーハードでは混乱弾を吐き出してきます。
＊メリクルが青色で難易度ノーマル、ハード、ベリーハードでは凍結弾を吐き出してきます
＊メリキュスが黄色で難易度ノーマル、ハード、ベリーハードでは感電弾を吐き出してきます
＊難易度アルティメットでは共通して即死弾を吐き出してきます。


＜ウル・ギボン / ゾル・ギボン＞
（Ul Gibbon / Zol Gibbon）
＊ウル・ギボンは難易度ノーマル、ハード、ベリーハードでフォイエを、アルティメットではギゾンデを吐き出してきます。
＊ゾル・ギボンは難易度ノーマル、ハード、ベリーハードでギバータを、アルティメットではメギドを吐き出してきます。
＊ゾル・ギボンの方がウル・ギボンよりも一回り大きいです。


＜シノワベリル / シノワスピゲル＞
（Sinow Berill / Sinow Spigell）
＊シノワベリルは難易度ノーマル、ハード、ベリーハードでギフォイエを、アルティメットではラフォイエを撃ってきます。
＊シノワスピゲルは難易度ノーマル、ハード、ベリーハードでレスタ、アルティメットではシフタ、デバンド、レスタを放ってきます。
＊シノワスピゲルは肩パーツの縁が赤色です。


＝＝＝＝＝＝＝プラント＝＝＝＝＝＝＝


＜ドルムオルム / ドルムダール＞
（Dolmolm / Dolmdarl）
＊ドルムオルムが青色です。
＊ドルムダールが紫色です。


＜シノワゾア / シノワゼレ＞
（Sinow Zoa / Sinow Zele）
＊シノワゾアが青色で難易度ノーマル、ハード、ベリーハードでギバータを、アルティメットではラバータを撃ってきます。
＊シノワゼレが赤色で全難易度共通でジェルン、ザルアを撃ってきます。


＜デルバイツァ＞
（Delbiter）
＊デルパイツァ（「ハ」に半濁点）
