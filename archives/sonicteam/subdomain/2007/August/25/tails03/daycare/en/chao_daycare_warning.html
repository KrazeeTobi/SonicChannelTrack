NOTES
Please read the following notes before visiting the Chao Daycare.

Notes when depositing your Chao.
  • You are able to deposit up to three Chao at one time.
  • You can choose the Chao to deposit from the available Chao Garden.
    Please be sure that the memory card is connected to the port with the Garden data in it.

Notes when picking up a Chao.
  • If a password is set on the Chao, input the password and then push the DOWNLOAD button.
  • If a program file (ex: CHAO ADVENTURE 2) exists in the memory card, you cannot pick up Chao. You must erase the program file or use another memory card.
  • Picking up a Chao needs 6 blank blocks in the memory card.
  • The Chao data which you picked up in memory card, will be erased automatically when you take the Chao to the Garden by the Chao Transporter.
