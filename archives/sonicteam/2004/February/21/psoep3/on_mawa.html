ネットワークブラウザの操作方法

ゲームキューブでホームページを閲覧する際における、ネットワークブラウザのパッド・キーボードの操作方法はこちらです。

■パッドの操作方法
GAMECUBE ブラウザ画面 ブラウザメニュー
方向キー カーソルの移動 フォーカス移動
Ａボタン 決定 決定
Ｙボタン ブラウザメニューを開く なし
Ｂボタン ブラウザメニューを閉じる メニューを閉じる
Ｘボタン なし なし
Ｃスティック 画面のスクロール なし
CTRLスティック マウスポインタの移動 マウスポインタの移動
Ｒトリガー 進む なし
Ｌトリガー 戻る なし
STARTボタン なし なし
B + X + START ソフトリセット ソフトリセット

■キーボードの操作方法
ブラウザ画面
キー 説明
F1 メニューバーを表示します
Enter フォーカスが当たっている項目を選択または実行します
Esc 選択を解除します
PageUp ページ単位で上にスクロールします
PageDown ページ単位で下にスクロールします
Ctrl + PageUp ページの1/10単位で、上にスクロールします
Ctrl + PageDown ページの1/10単位で、下にスクロールします
Home ページの先頭を表示します
End ページの最後を表示します
← ポインタを左に移動します
→ ポインタを右に移動します
↑ ポインタを上に移動します
↓ ポインタを下に移動します
Shift + ← ページの1/10単位で、左にスクロールします
Shift + → ページの1/10単位で、右にスクロールします
Shift + ↑ ページの1/10単位で、上にスクロールします
Shift + ↓ ページの1/10単位で、下にスクロールします
B ブラウザの一つ前のページに戻ります
F ブラウザの一つ後のページに進みます
Shift + Q ブラウザ画面を終了します
Shift + F フレームを切り替えます
Ctrl + Shift + F 逆順でフレームを移動します
Shift + R 再読み込みします
Back Space ブラウザの一つ前のページに戻ります
Tab 選択項目またはリンクのフォーカスを次に移動します
Shift + Tab 選択項目またはリンクのフォーカスを前に移動します

ブラウザメニュー画面
キー 説明
F1 メニューを閉じます
Enter フォーカスが当たっている項目を選択または実行します
Esc メニューを閉じます
← ポインタを左に移動します
→ ポインタを右に移動します
↑ ポインタを上に移動します
↓ ポインタを下に移動します
Tab 選択項目またはリンクのフォーカスを次に移動します
Shift + Tab 選択項目またはリンクのフォーカスを前に移動します
