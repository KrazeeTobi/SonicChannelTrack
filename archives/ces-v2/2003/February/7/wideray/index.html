2003 International CES Handheld Show Guide

At Your Fingertips

Install the 2003 International CES Handheld Show Guide today, and walk the show floor powered with our exhibitor directory and show floor information. Add conference sessions and keynotes to your datebook, so you don't miss a thing! Exhibitor and conference data is as of 01/02/03.

Begin installation by selecting the appropriate 'Download' button. You will be prompted to download a '.zip' file containing two files. Having done so, follow the simple instructions below.

Pocket PC Users:
System Requirements:
  • Microsoft ActiveSync software installed on desktop
  • Pocket PC 2000 or 2002 device
  • 525K or more available memory on device


Instructions:
1. With your Pocket PC mobile device connected to your computer, double-click 'WideRay Setup.exe.' This will launch the Add/Remove Programs application for your mobile device. A dialog will appear: "Install WideRay Viewer?" Click 'Yes.'
2. When installation is complete, the WideRay Viewer application will be open on your Pocket PC. CLOSE THE VIEWER by tapping the 'View' menu in the lower left corner of the screen and selecting 'Exit.'
3. Having closed the viewer application, double-click 'CES 2003.exe' to install the CES Show Guide.
4. When installation is complete, re-open the WideRay viewer. To open, select the WideRay icon in your Start menu. Then, tap 'View' in the lower left corner of your screen and select 'CES 2003.'
Palm Users:
System Requirements:
  • Palm HotSync software installed on desktop
  • Palm OS 3.1 or later device
  • 450K or more available memory on device



Instructions:
1. Double-click each file: 'WideRay.prc' and 'CES_2003.PDB.' This will launch the Install Tool application, which will automatically prepare the files for installation.
2. Click 'Done' in the Install Tool. A dialog may appear, informing you that the files will be installed during the next HotSync operation. Click 'OK.'
3. With your Palm mobile device connected to your computer, perform a HotSync to install the files.
4. When HotSync has finished, select the WideRay icon from the main menu on your Palm device.

Save the date!
