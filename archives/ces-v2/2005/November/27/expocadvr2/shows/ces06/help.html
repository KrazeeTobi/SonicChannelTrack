Internet Explorer users having trouble loading the floor plans?

If you do not have Java enabled in your Internet browser, you may experience problems loading the floor plans. Follow these brief instructions to ensure Java is enabled on your browser.
  1. In Internet Explorer, go to Tools > Internet Options. Click on the Advanced Tab.
  2. Scroll to the section entitled Microsoft VM.
  3. Confirm that Java console enabled and JIT compiler for virtual machine enabled are checked (may require reboot).
  4. If you do not have a Microsoft VM section, or if it is entitled Java VM, you will need to install the latest version of the Java Virtual Machine. To do this, go to http://www.java.com/en/download.
  5. Restart your computer before reloading the floor plans.
