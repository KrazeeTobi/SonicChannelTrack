64MB IC Memory Stick™ Digital Data Storage and MagicGate™ Memory Stick
The 64MB IC Memory Stick media is an ultra-compact storage media designed to capture and share digital data, including approximately 42 minutes of MPEG movies (Video Mail mode) in one minute intervals, or more than 1,000 still images (640 x 480 resolution). The 64MB MagicGate Memory Stick media can store up to 80 minutes of secure digital music files – whether from the Internet or from your personal CD collection – for playback on Sony’s Memory Stick Walkman® personal stereo.
