●火事だ！地震だ！ ソニックと一緒に防災訓練！ 2007年9月27日

新ブログURLはコチラ☆ → http://blog.sonic-channel.jp/

みなさんこんにちは！　
ソニックチャンネル スタッフ茂呂です。

昨日はセガ本社で、防災訓練がありました！

「○○階の給湯室で、火災が発生いたしました。」

「アナウンスに従って、指示された階より避難を開始して下さい。」

我が社では”非常持出袋”が１人１セットづつ配られており、
皆それをしょっての避難となります。


ええ。

私はモチロン、ソニックもしょって。


●消火訓練

なんかへっぴり腰ですが、私も消火訓練中でございます。
みんなで順番に、全員で練習しました！

もう消火器もこわくありません！


●地震体験

今年は地震体験車まできました！
こちらも順番に全員で体験しました！

震度７は、そりゃーもー、こりゃーテレビもタンスも吹っ飛んでくるわなぁというくらいなものすごい揺れでした！

我が家の安全確認もあらためて行なわなければと思い、身が引き締まる思いでした。


●ソニックも大地震体験中

アップにするとこんな感じです。
神妙な面持ちで体験しておりました。


みなさんも、今一度身の回りの安全確認を！

●TGS来場者193,040人！ 2007年9月26日

新ブログURLはコチラ☆ → http://blog.sonic-channel.jp/

みなさんこんにちは！　
ソニックチャンネル スタッフ茂呂です。

東京ゲームショウ2007がありました先週末は、秋分の日、お彼岸でもありましたね。
”暑さ寒さも彼岸まで”と言いますが、まさにそんな感じでした。
（わが家では、おはぎ作りをしました。）

先々週の敬老の日には老人を敬愛し、長寿を祝う。

先週の秋分の日には祖先を敬い、なくなった人々をしのぶ。

ついついお休みだけに浮かれそうになってしまいましたが、
本来の意味も忘れないでおきたいですね。


TGSですが、今年は193,040人もの来場者があったそうです。
すごい数ですよね！
参加されたみなさんお疲れ様でした！

『日本ゲーム大賞２００７』未発売作品を対象とする「フューチャー部門」では、
セガからは『龍が如く 見参！』が受賞作品に選ばれました。


ソニックタイトルも、これからガンガンがんばっていきますよっ！


ダウンロードアイテム盛りだくさん♪

●週末はTGSへ♪ 2007年9月22日

新ブログURLはコチラ☆ → http://blog.sonic-channel.jp/

みなさんこんにちは！　
ソニックチャンネル スタッフ茂呂です。

いよいよ、本日から東京ゲームショウ2007一般公開になりますーーー！


ソフトバンクモバイルブースでは、
「対戦ぷよぷよ通」、「ぷよぷよフィーバー デラックス」両コンテンツをプレイしてくれた方に、
「ぷよぷよコンパクトミラー」をプレゼント☆

９月２３日（日）１６：００からは
Ｈｉｒｏ師匠、光吉猛修を中心としたＳＥＧＡサウンドユニット[Ｈ．]のライブも開催予定♪

セガブースでも、連日色々なステージイベントを開催しています☆
詳細等は、
TOKYO GAME SHOW 2007 SEGA OFFICIAL WEBSITE
http://tgs.sega.jp/
からご覧下さい♪

「TGSに行けないよ〜〜〜！」

という方は、こちらのサイトで
ステージイベントのライブ中継も行っていますので、是非ご自宅からご覧になって、臨場感を感じ取ってもらえればと思います☆

家に居ながらも、十分楽しんでいただけるかと思います♪


それではみなさん、TGSを、会場から、ご自宅から、どうぞお楽しみ下さいー！＾＾
