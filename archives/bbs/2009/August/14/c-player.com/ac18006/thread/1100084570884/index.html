http://streaming.yahoo.co.jp/
　[会員登録]
　　必要(無料)
　[購入単位 / 価格 / 視聴期間]
　　4話 / 336円 / 7日
　[支払方法]
　　クレジットカード、もしくは口座振替
　　　別途Yahoo!ウォレットへの登録が必要(無料)


【DMM】
http://www.dmm.com/
　[会員登録]
　　必要(無料)
　[購入単位 / 価格 / 視聴期間]
　　1話 / 105円 / 7日
　[支払方法]
　　クレジットカード
　　銀行振込
　　BitCash EX
　　Edy
　　コンビニ振込


【あにてれシアター】
http://id.ani.tv/
　[会員登録]
　　必要(無料)
　[購入単位 / 価格 / 視聴期間]
　　1話 / 105円 / 48時間
　[支払方法]
　　クレジットカード
　　銀行振込
　　Edy
　[備考]
　　第2期配信開始が1日遅れの8月7日


【ShowTime】
http://www.showtime.jp/genre
　[会員登録]
　　必要(月額 294円)
　[購入単位 / 価格 / 視聴期間]
　・第1話
　　　無料 / 無制限
　・第2話〜第25話
　　　6話 / 504円 / 購入から7日

　　　7話 / 588円 / 購入から7日
　　　　もしくは
　　　1話 / 105円 / 購入から7日
　[支払方法]
　　クレジットカード
　　銀行振込
　　WebMoney
　　Edy
　　BitCash
　[備考]
　　第2期配信開始が1日遅れの8月7日

1話無料、他まとめ買いで84円/話。
ただし、それとは別に月額294円が必要なので注意。

月額無料が良いなら「Yahoo!動画」でまとめ買いが一番お得。(84円/話)
