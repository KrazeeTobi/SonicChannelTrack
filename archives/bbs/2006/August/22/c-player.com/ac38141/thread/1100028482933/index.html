●HAPPY BIRTHDAY SONIC!! 2006年6月23日

みなさんこんにちは！　
SONIC CHANNEL スタッフ茂呂です。


みなさん、
いつも応援、ご声援ありがとうございます！


２００６年６月２３日（金）
ソニックは１５周年目のバースデイを迎えることが出来ました！



「ソニック チャンネル」では、ソニック１５周年記念特別更新をいたしました！
< http://sonic.sega.jp/ >



・[イベント/人気投票結果発表]：キャラクターランキング発表！
・[キャラクター/年表]：
　ソニック誕生から１５年の歴史を一挙公開！
・[おたのしみ/かべがみ]：
　ソニック１５周年記念特別かべがみ追加！
・[おたのしみ/ペーパークラフト]：
　「グリーンヒル」のペーパークラフト！

・ソニック生誕15周年の記念グッズを、
セガダイレクト限定で発売決定！
（06/23 13時からの販売となります。）

・『PSO BB』のビジュアルロビーに
ソニックとテイルスが登場！


お待ちかね、キャラクター人気投票の結果発表も掲載！

そして！

兼ねてからご要望の多かった、「ソニック年表」も新たに掲載！！

「１５周年記念特別かべがみ」はもちろん、
「ペーパークラフト」も追加！！

ソニック１５周年の歴史を、振り返りながら
楽しめる要素いっぱいの「ソニック チャンネル」です♪

週末は、「ペーパークラフト」作りで忙しくなるかな？！＾＾


ソニック、お誕生日おめでとう！！
みなさんも、本日は一緒にお祝いして下さいね♪♪＾＾



ソニック１５周年記念特別更新！
ソニック公式ポータルサイト「ソニック チャンネル」
< http://sonic.sega.jp/ >
