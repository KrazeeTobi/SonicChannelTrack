[594] ジェットのミッションについて（ネタバレ）

From： カイリ

どうもお久しぶりのカイリです。
これは、とあるサイトから仕入れた情報ですが、
ジェットのミッションの出現条件はストームとウェーブのミッションを全てクリアする事です。
ランクは関係ありません。（オール金にすると超特殊なギアが手に入ります。）
また、隠しステージのミッションもクリアする必要があります。
殆どのステージはストーリーモードで手に入りますが、二つだけはワールドグランプリで優勝しなければなりません。
皆さん頑張りましょう。
