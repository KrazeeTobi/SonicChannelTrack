テーブルの作り方

<table>
</table>
とりあえずこれが基本形です。


<table border="数字">
<tr><td>ここに文字を書く</td></tr>
</table>

と書くと
|~~~~~~~~~~~~~~~~|
|ここに文字を書く|
|________________|
こういうのができます
border="ここに入れた数字によって高さが変わります"
枠を表示させたくない時はborderを書かないでください。
<tr>は行、<td>は列です。<td>の代わりに<th>にすると中の文字が太字の中央揃えになります、でもＤＰではどちらも同じように見えます。


<tabre border cellpadding="数字">
<tr><td>　　</td><td>　　</td></tr>
<tr><td>　　</td><td>　　</td></tr>
</tabre>

と書くと
|~~~~~|~~~~~|
|_____|_____|
|　　 |　　 |
|_____|_____|
と表示されます
cellpadding="ここに入れた数字の分だけテーブル内に余白ができます"


<tabre border bgcolor="色">
<tr><td rowspan="数字（今は２とする）">　　</td><td colspan="数字（今は２とする）">　　</td></tr>
<tr><td>　　</td><td>　　</td></tr>

と書くと
|~~~~~~|~~~~~~~~~~~~~|
|　　　|_____________|
|　　　|　　　|　　　|
|______|______|______|
と表示されます
td rowspan="ここに入れた数字の分だけ縦にくっつきます"
td colspan="ここに入れた数字の分だけ横にくっつきます"
bgcolor="ここに色を書くとテーブル全体の色が変わります"

とりあえずこんなものですね、でもこんな書き方で解ったかな・・・。
他にも色々ありますけど、知りたければどんなことがしたいのかレスをください。
